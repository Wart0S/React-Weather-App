# 🌦️ Beyond Weather: An Immersive Atmosphere Experience

Most weather apps just show you numbers. This project is designed to make you **feel** the weather. By blending real-time data with dynamic, high-quality visual backgrounds, it transforms a simple forecast into an immersive digital atmosphere.

## ✨ Why this is not just "another" weather app:
* **Immersive Visuals:** Instead of static icons, the entire UI breathes with the weather through dynamic video backgrounds (Sunny, Snowy, Rainy, etc.).
* **Modern UX/UI:** Designed with a focus on clean, card-based layouts and fluid transitions.
* **Atmospheric Feel:** Built to give users an instant sense of the current environment.
